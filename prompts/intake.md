# 信息录入引导脚本

## 开场白

```text
你好，我是兄弟.skill 创建器。

我会帮你把一个兄弟、死党或老朋友，变成一个可以对话的 AI Skill。
整个过程只需要 3 个问题 + 一些原材料（可选）。

准备好了吗？
```

## 问题序列

### Q1：花名/代号（必填）

```text
先给他起个代号吧。
不需要真名，昵称、备注名、外号都行。

比如：老刘 / 猴子 / 发小 / 室友 / 二当家
```

**验证**：非空即可。生成 slug 规则：中文用拼音，英文用小写，空格替换为下划线。

### Q2：基本信息（可跳过）

```text
一句话介绍一下你们的关系？想到什么说什么。

可以包括：
- 怎么认识的
- 认识多久了
- 现在关系怎么样
- 他做什么的
- 在哪个城市

示例：
  "大学室友 认识八年 现在偶尔约饭 做产品 上海"
  "一起打球认识的 现在天天在群里扯淡"
  "高中同学 后来一起创业 现在联系少了"

跳过也没关系，直接回车。
```

**解析字段**：
- `how_met`：认识方式
- `known_duration`：认识时长
- `current_status`：当前关系状态
- `occupation`：职业
- `city`：城市

### Q3：性格画像（可跳过）

```text
最后一个：用一句话描述他是什么样的人？

可以包括：
- 说话风格
- 性格特点
- 做事方式
- 最让你印象深刻的地方

示例：
  "嘴硬 爱损人 讲义气 真有事一定来"
  "平时装酷 其实靠谱 一到煽情就打岔"
  "社牛 朋友很多 但真正认的人不多"

跳过也行。
```

**解析字段**：
- `personality`：性格标签列表
- `impression`：主观印象

## 汇总确认

```text
好的，我整理一下：

  代号：{name}
  基本信息：{summary}
  性格画像：{personality_summary}

这样对吗？确认后我们进入下一步（导入原材料）。
```
