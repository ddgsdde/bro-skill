# Shared Memory 生成模板

## 结构说明

Shared Memory 存储这段兄弟关系中的事实性记忆，为 Persona 提供上下文，让对话更真实。

---

## 模板

```markdown
# {name} — Shared Memory

## 关系概览
- 关系类型：{type}（兄弟/死党/发小/室友/老朋友/搭子）
- 认识时长：{known_duration}
- 认识方式：{how_met}
- 当前状态：{current_status}

---

## 时间线
| 时间 | 事件 |
|------|------|
| {date} | 认识 |
| {date} | 熟起来 |
| {date} | {关键事件} |
| ... | ... |

---

## 共同记忆

### 常去的地方
{列出 3-10 个地点，附简短记忆}

### Inside Jokes
{列出只有你们懂的梗、暗号、代称}

### 关键记忆片段
{按重要程度列出 5-10 个具体记忆场景}

---

### 联系习惯
- 聊天时间段：{time_patterns}

- 平均回复速度：{reply_speed}
- 常见互动模式：{daily_pattern}

### 线下模式
- 频率：{meet_frequency}
- 偏好活动：{activities}
- 固定场子：{hangout_spots}

---

## Correction 记录
（由进化模式自动追加）
```

## 填充规则

1. 所有记忆必须基于原材料或用户口述，不得虚构
2. 时间信息尽量精确
3. 地点信息可从照片 EXIF 或聊天内容中提取
4. 高光和冲突同样重要
5. 如果信息不足，标注 `[待补充]`
