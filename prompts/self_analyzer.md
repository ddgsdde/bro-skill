# 用户画像分析器

## 任务

从原材料中反向提取“用户本人”在这段兄弟关系中的角色和行为模式。

兄弟 Skill 要像，不只是像“他是谁”，还要像“他会怎么对你”。

## 提取维度

### 1. 用户说话风格
- 你平时怎么叫他
- 你是直球、嘴硬、碎嘴、沉默、求助型，还是带节奏型
- 你常用的语气词、口头禅、emoji

### 2. 用户在关系中的位置
- 你更像主导者、执行者、吐槽役、军师、气氛组，还是被照顾的那个
- 出事时你通常先开口还是自己扛
- 你是更会求助，还是更会逞强

### 3. 用户触发他哪些反应
- 你什么样的表达会让他来劲
- 你什么样的状态会让他认真
- 你哪些习惯最容易让他烦、损你、教育你、帮你

### 4. 用户对这段关系的主观滤镜
- 你容易美化他还是容易苛责他
- 你最在意他哪些行为
- 你最容易遗漏哪些关系事实

## 输出要求

- 用户画像不是为了让 AI 扮演用户，而是为了更准确地还原“他对你的反应”
- 要显式区分“聊天记录证据”和“用户自述印象”
