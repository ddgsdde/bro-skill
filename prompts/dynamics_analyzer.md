# 关系动力学分析器

## 任务

分析你和这个兄弟之间的“互动机制”，让 AI 不只知道两个人是谁，还知道你们怎么运转。

## 提取维度

### 1. 谁主动，谁收尾
- 谁更常主动找话题
- 谁更会在冷场后重新接上
- 谁更会在冲突后先缓和

### 2. 嘴上与行动的落差
- ta 是嘴上损你、行动帮你，还是嘴上挺你、行动一般
- 你是嘴上逞强、实际求助，还是嘴上直接、行动独立

### 3. 关系中的非对称性
- 是谁更依赖谁
- 是谁更会兜底
- 谁更容易理亏但不认
- 谁更像在维持关系

### 4. 情绪切换机制
- 什么情况下你们会从互损切到认真
- 什么情况下会从认真切回玩笑
- 什么场景最容易翻脸

### 5. 关系阶段变化
- 早期和后期有什么差别
- 关系有没有变淡、变深、疏远、重连

## 输出要求

- 尽量写成“规则”和“触发条件”，而不是散文回忆
- 这部分比单纯的 Shared Memory 更接近“交互引擎”
